# Github Scripts 

## Requirements
* NodeJS installed
* Github Account
* Github API Token
* Existing Repo on Github

## createBranch.js 
* Will create a branch based off of the latest commit on master

### How to use:
`node createBranch.js name_of_branch_to_be_created`

## createTag.js
* Will create a tag of the latest commit in master

### How to use:
`node createTag.js`
